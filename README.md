# Food-Counter-System

Implemented using Data Structures in C

Used Data Structures like Stacks , Queues, Linked Lists

Implemented Authentication for secure login

Project implemented in 2 Views:
  1. Admin
  2. User/Customer

Output : 

I. User View:
    1. Order
    2. View Cart
    3. Proceed to Checkout
    0. Back
  
II. Admin View:
    1.Edit Menu
        a. Add item
        b. Edit item
        c. View Menu
        d. Back
    2.Orders
        a. Serve Order
        b. View Orders
        c. Back
    3.Back
